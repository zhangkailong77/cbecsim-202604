# 开局资金建议说明（第二步配套）

## 结论
- 系统默认初始资金：`200000 RMB`
- 说明：该数值更适合教学场景，能覆盖选品、备货、物流、仓储、推广与回款周期带来的资金占用。

## 真实跨境店铺首期成本区间（参考）
- 轻量试水：`2万-5万 RMB`
- 稳定起店：`5万-12万 RMB`
- 稍规模化：`12万-30万 RMB`

## 典型成本构成（参考占比）
- 首批备货：`40%-60%`
- 国际物流与清关：`15%-25%`
- 海外仓（仓租/操作费）：`10%-20%`
- 推广与活动：`10%-20%`
- 流动资金缓冲：至少 `1-2` 个月

## 在教学仿真中的应用
- 使用 `200000` 作为默认资金，可降低新手在前几轮因现金流断裂而“无法继续”的概率。
- 同时保留可配置能力，后续可扩展难度档位（如 30000 / 100000 / 200000）。
