# CBEC SIM 阶段设计文档（第二步）

## 阶段进度
- 第一步（已完成）：注册/登录功能
- 第二步（当前）：开局准备功能设计

## 第二步目标
- 玩家登录后进入“开局准备”页面。
- 确认初始资金、目标市场、经营周期。
- 点击“开始经营”后创建当前局数据，进入后续经营页面（Shopee 运营）。

## 开局准备（V1）字段
- `initial_cash`：初始资金（默认 `200000`，可配置）
- `market`：目标市场（首版先支持 `MY`）
- `duration_days`：经营周期天数（默认 `365`）

## 核心流程
1. 玩家完成登录。
2. 系统判断是否存在进行中的经营局。
3. 若无进行中局，进入“开局准备”页。
4. 玩家确认参数并点击“开始经营”。
5. 系统创建一局经营数据并进入 Shopee 模块。

## 数据与状态（V1）
- 一名玩家同一时间仅允许一个进行中局（`running`）。
- 开局数据需持久化，支持刷新后恢复。

## 本阶段暂不包含
- 多人对战
- 随机突发事件（运费波动、关税变化等）
- 员工管理

## 验收标准
- 登录后能正确进入开局准备页。
- 提交开局参数后能成功创建经营局。
- 创建成功后自动进入经营页面。
- 刷新后能保留当前进行中的经营局状态。
