# 最终版 02：经济系统与财务模型

## 1. 订单生成模型
`订单量 = 基础流量 * 曝光系数 * CTR * CVR * 价格竞争系数 * 店铺健康系数 * 活动系数`

参数说明：
- 基础流量：由品类景气、节庆、平台流量池决定
- 曝光系数：由广告出价、店铺权重决定
- CTR：主图/价格吸引力（抽象）
- CVR：评价、时效、价格、客服响应共同决定

## 2. 单笔利润模型
`净利润 = 销售收入 - 商品成本 - 头程物流 - 本地仓配 - 平台费用 - 支付费用 - 广告分摊 - 售后损耗 - 税费 - 财务费用`

## 3. 现金流模型
- 入：订单回款（有延迟）
- 出：采购付款、物流付款、广告扣费、人工与仓租

`现金安全天数 = 可用现金 / 近7日平均净流出`

阈值建议：
- `<14天`：黄灯
- `<7天`：红灯（触发融资/收缩建议）

## 4. 汇率模型
`当日汇率 = 趋势项 + 随机扰动 + 事件冲击`

`外币成本(MYR) = 外币成本 * 当日汇率 * (1 + 换汇费率)`

## 5. 政策参数中心（关键）
由于平台费率与政策可能更新，建议全部参数可配置化：
- 平台交易手续费率
- 单笔服务费
- 分期支付费率
- SST附加
- LVG税率

> 游戏不写死费率，采用“政策版本包”机制，支持热更新。

## 6. 默认参数建议（首发可调）
- 退款率：2%~12%（按品类）
- 丢损率：0.1%~1.2%（按物流方案）
- 广告CTR：0.8%~3.5%
- 广告CVR：1.0%~6.0%
- 库存安全天数：7~21天

## 7. 财务报表输出
- 日报：收入、成本、广告、库存变化
- 周报：利润表 + 现金流量表 + SKU贡献榜
- 赛季报：经营评分与策略复盘建议
